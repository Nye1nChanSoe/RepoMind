## Product Definition

RepoMind is a context-aware code change assistant.

It helps a developer move from:
- a repository URL
- a natural-language change request

to:
- the most relevant code
- a plain-English explanation of current behavior
- a minimal implementation plan
- a proposed diff with explanation

## Core Promise

RepoMind does not promise to understand an entire codebase.

It promises something narrower and more useful:
- find the code that most likely matters
- reason over that code in a transparent way
- propose a small, reviewable change

## Product Category

RepoMind sits between:
- code search and retrieval tools
- fully autonomous coding agents

It should feel more helpful than retrieval alone, but more controllable than a general autonomous agent.

## Product Character

RepoMind should be:
- scoped
- inspectable
- practical
- low-friction
- inexpensive enough to use repeatedly
