# /docs/plans/02_milestones.md

## Milestone 1: Repository Intake Works

Definition:
- can clone a repository
- can discover supported files
- can skip obvious noise directories

## Milestone 2: Retrieval Is Useful

Definition:
- can chunk and embed a repository
- can retrieve code that a human agrees is relevant for simple requests

## Milestone 3: Pipeline Is End-to-End

Definition:
- understand, plan, and generate all run successfully
- final output is structured and inspectable

## Milestone 4: UI Demo Is Stable

Definition:
- Streamlit flow works for multiple sample repos
- diffs and explanations render cleanly

## Milestone 5: Demo Readiness

Definition:
- tested on at least three repositories
- known limitations are documented
- environment setup is repeatable
